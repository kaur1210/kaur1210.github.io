List of Html tags

Paragraph
For adding a paragraph of text we can use the (p) tag. The browser will add a space before and after the content of the paragraph.
Quote
a small quotation embedded within other content. "You become what you think"
Heading
We can use the (h1)tag for the most important heading of the page. For the next level of heading we have (h2),(h3,(h4)),(h5) and (h5) The browser will style each heading differently, stating with the biggest size for h1 then h2 and so on.
Emphasis
a string of emphasized, slightly more important text. Screen readers will change their voice for this text.
Technical term
defines technical term, a ship name, a book title, a thought, sarcasm, another languages
